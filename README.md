\# 接口自动化测试项目

基于 Python + Unittest + Requests 实现的接口自动化测试框架



\## 📌 项目介绍

本项目用于接口自动化测试实战，支持：

\- 读取 JSON 测试数据

\- 参数化执行用例

\- 接口请求封装

\- 统一测试报告

\- Git + GitHub 版本管理



\## 📁 项目结构

api\_auto\_test/├── api/           

\# 接口请求封装├── case/           

\# 测试用例├── data/           

\# JSON 测试数据├── tools/          

\# 工具类（读取 JSON 等）





\## 🚀 技术栈

\- Python 3.x

\- Unittest 测试框架

\- Requests 接口请求

\- JSON 数据驱动

\- Git + GitHub 版本控制



\## 🎯 核心功能

\- 登录接口自动化测试

\- JSON 文件参数化数据

\- 测试结果自动断言

\- 代码云端托管（GitHub）



\## 📌 测试场景

\- 正确账号密码 → 登录成功

\- 错误账号 → 登录失败

\- 空参数 → 登录失败
